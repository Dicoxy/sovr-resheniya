## О проекте
Сайт дистрибьютора промышленных роботов. Миграция с Битрикса на Next.js.
Стиль: тёмный, технологичный, high-tech (похоже на systech-team.ru).

## Методология
**Читай перед работой:** https://github.com/Dicoxy/vibecoding-methodology
- QUICKSTART.md — фазы работы
- patterns/ — решения типовых задач
- components/ — готовые компоненты

## Стек
- Next.js 14 (App Router)
- TypeScript  
- Tailwind CSS
- Framer Motion (анимации)
- Docker (деплой)

## Структура секций (план)
```
1. Header — навигация + каталог
2. Hero — слайдер брендов + видео-эффект
3. Catalog — сетка категорий роботов
4. Partners — логотипы брендов
5. ROI Calculator — калькулятор окупаемости (новое)
6. MARS Showcase — витрина системы управления (новое)
7. Media — новости
8. About — о компании
9. Contact — форма заявки
10. Footer
```

### AnimatedLines — линии к карточкам
```tsx
<AnimatedLines
  containerRef={containerRef}
  originRef={originRef}
  targetRefs={[ref1, ref2, ref3]}
  colors={['#00ff88', '#3b82f6', '#f59e0b']}
  isInView={isInView}
  pattern="fan"
/>
```

⚠️ **НЕ анимировать x/y целевых элементов** — только `opacity`

## Особенности проекта

### Hero с видео-эффектом
1. Сначала статичный баннер с роботом
2. Плавный fade-out картинки
3. "Проступает" затемнённое YouTube-видео Pudu
4. Видео играет приглушённо на фоне

### ROI-калькулятор
Интерактивный расчёт окупаемости роботов:
- Выбор типа бизнеса
- Площадь/объём работ
- Сравнение с ручным трудом
- Срок окупаемости

### Каталог товаров
- ~20 товаров
- 7 категорий
- JSON-данные (не CMS)
